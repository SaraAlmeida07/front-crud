# 📝 JS CRUD Project

Uma aplicação frontend modular em JavaScript para gerenciar usuários (Create, Read, Update, Delete). Este projeto demonstra a aplicação de uma arquitetura limpa com separação de responsabilidades (API, DOM e Controle) e foi conteinerizado para rodar de forma isolada.

## 🛠️ Tecnologias Utilizadas

* **HTML5 & CSS3:** Estrutura e estilo base.
* **JavaScript (ES6+):** Lógica da aplicação.
* **Bootstrap 5:** Estilização e sistema de grid responsivo.
* **Axios:** Consumo da API (métodos GET, POST, PUT, PATCH e DELETE).
* **Docker & Nginx:** Servidor web leve e conteinerização do projeto.

## ⚙️ Pré-requisitos

Para rodar este projeto, você precisará ter o [Docker](https://www.docker.com/) instalado na sua máquina.

## 🚀 Como Executar o Projeto

1. Clone este repositório para a sua máquina:
   ```bash
   git clone <URL_DO_SEU_REPOSITORIO_AQUI> 

2. Acesse a pasta raiz do projeto pelo terminal
    ```bash
    cd <NOME_DA_SUA_PASTA_AQUI> 
3. Suba o container com Docker Compose

4. Acesse no navegador
